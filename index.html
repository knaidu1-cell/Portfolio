SAN FRANCISCO AIRPORT PERFORMANCE ANALYSIS USING GOOGLE CLOUD PLATFORM

The purpose of this project was to analyze San Francisco International Airport’s (SFO) performance by building a data pipeline on Google Cloud Platform (GCP). The goal was to extract, transform, and analyze airport passenger traffic data to gain insights into airline performance, passenger volumes, and regional travel trends. By automating the data collection and analysis process, the project aimed to improve operational efficiency, assist in resource planning, and provide actionable insights for airport management and airlines. Key analyses included identifying the top-performing airlines, passenger traffic trends, and regional travel patterns, which could help optimize flight scheduling, staffing, and infrastructure planning. The insights were visualized using Looker Studio dashboards, enabling data-driven decision-making for better airport management and enhanced passenger experience.

TEXT ANALYSIS USING DEEP LEARNING

The purpose of this project was to analyze McDonald’s Franchise Agreement using Natural Language Processing (NLP) techniques to gain insights into franchisee rights, territorial competition, and corporate control. By applying text analysis, the study aimed to uncover key themes related to branding, territorial restrictions, and franchise operations. The project explored how McDonald’s controls its franchise network, whether franchisees receive exclusive territories, and how competition from both company-owned outlets (McOpCo) and third-party franchises impacts them.
